SigFREE

Signature Free Buffer Overflow Attack Blocker

What is SigFree?

SigFree is a buffer overflow attack blocker, which is signature free. It's based on a paper by Xinran Wang and etal.

Prerequisites

graphviz mongodb php5 apache2 udis86 ffi ffi-udis86 graph colorize json mongo

Usage

You can run sigfree as ruby proxy.rb If you want automated test. Run test.rb from /test ruby test.rb If you want a log of inputs, place ui in your apache's www folder
